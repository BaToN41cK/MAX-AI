# MAX-AI

Консольный AI агент на Python, использующий LLM модели Cohere и Mistral. Поддерживает чтение веб-страниц по ссылкам прямо в тексте запроса.

## Что это за проект

MAX-AI - это CLI инструмент, который позволяет взаимодействовать с искусственным интеллектом через терминал. Особенность - автоматическое извлечение и чтение URL из запросов.

### Зачем нужен и чем полезен

- **Быстрый доступ к AI**: Не нужно открывать браузер - получайте ответы сразу в терминале
- **Анализ веб-контента**: Автоматически загружает и аналирует веб-страницы, PDF, DOCX файлы и YouTube видео прямо в запросе
- **Контекстное понимание**: Можно задавать вопросы о содержимом веб-страниц, не переключаясь между приложениями
- **Кеширование**: Повторные запросы мгновенно отвечаются из кеша, экономя время и API лимиты
- **История запросов**: Все запросы сохраняются для повторного использования и анализа
- **Автономность**: Работает офлайн (чтение URL требует интернет, но не требует посещения сайтов вручную)
- **Автоматическая суммаризация**: Большие документы (>40K символов) автоматически суммаризируются для оптимального использования контекста

## Как это работает

1. **Извлечение URL**: Регулярным выражением `https?://\S+` находятся все ссылки в тексте запроса
2. **Загрузка страниц**: Через `requests` скачивается HTML, `BeautifulSoup` очищает от script/style и возвращает текст
3. **Обработка специальных форматов**:
    - PDF: Извлечение текста через PyPDF2
    - DOCX: Извлечение текста через python-docx
    - PPTX: Извлечение текста через python-pptx
    - XLSX/XLS: Извлечение текста через openpyxl и xlrd
    - TXT/MD: Чтение как обычный текст
    - YouTube: Получение транскрипта через youtube-transcript-api (если установлен)
4. **Объединение**: Содержимое страниц добавляется к запросу как контекст
5. **История диалога**: Последние N сообщений из истории диалога добавляются в промпт для лучшего понимания контекста
6. **Отправка в Cohere**: Запрос отправляется в модель `command-a-03-2025` для получения начального ответа
7. **Обработка Mistral**: Если доступен ключ Mistral, начальный ответ отправляется в модель Mistral для модернизации и дополнения правил
8. **Кеширование**: Ответ сохраняется в `~/.max_ai_cache.json` (по умолчанию 1 час)
9. **История**: Все запросы сохраняются в `~/.max_ai_history.json`

## Features

- Автоматическое извлечение и чтение URL из запроса (регулярное выражение `https?://\S+`), включая PDF, DOCX, PPTX, XLSX, XLS, TXT, MD файлы и транскрипты YouTube
- Интеграция с Cohere (основная модель)
- Поддержка Mistral (дополнительно, для улучшения ответов)
- Кеширование результатов (TTL по умолчанию 1 час)
- История запросов с таймстампами
- Работа на русском языке
- Извлечение текста из PDF, DOCX, PPTX, XLSX, XLS, TXT, MD файлов и YouTube видео
- **Enhanced CLI с Rich:**
   - Прогресс-бар при загрузке нескольких URL
   - Анимированный спиннер при получении ответа
   - Вывод времени выполнения и токенов в реальном времени
   - Цветные сообщения о прогрессе
- **Reliability improvements:**
   - Автоматический retry с exponential backoff (3 попытки, задержка: 0.5s → 1s → 2s)
   - Валидация URL перед загрузкой
   - Rate limiting (максимум 5 одновременных запросов)
   - User-Agent для обхода 403 Forbidden ошибок
- **Token tracking:**
  - Реальное количество токенов из API ответа
  - Стоимость запросов в USD
- **Статистика использования:**
  - Количество запросов за день/неделю
  - Суммарные токены и примерная стоимость
  - Частота использования моделей
  - Популярные домены (источники контента)

## Настройка PATH (Windows)

**Автоматическое добавление:** При установке через pip скрипт автоматически добавляет путь к `max-ai.exe` в системный PATH.

После установки, если команда не работает, выполните один из вариантов:

### Вариант 1: Добавить в PATH через графический интерфейс

1. Панель управления → Система → Дополнительные параметры системы
2. Переменные среды → Path → Изменить → Добавить

**Добавьте путь (узнайте точный путь в выводе pip install):**
```
# После установки найдите строку вида:
# "max-ai.exe is installed in '...'"
# Вставьте этот путь
```

Для Python 3.13 обычно:
```
%APPDATA%\Python\Python313\Scripts
%USERPROFILE%\AppData\Roaming\Python\Python313\Scripts
```

### Вариант 2: Использовать напрямую

```bash
python -m max_ai.cli run "ваш запрос"
```

## Настройка

Создайте файл `.env` в директории проекта или добавьте переменные окружения:

```bash
COHERE_API_KEY=your_cohere_api_key
MISTRAL_API_KEY=your_mistral_api_key  # опционально
```

## Конфигурация (max-ai.yaml)

Создайте файл `max-ai.yaml` в директории проекта для настройки:

```yaml
cohere_model: "command-a-03-2025"      # Модель Cohere
mistral_model: "mistral-large-latest"    # Модель Mistral
max_content_length: 50000               # Максимальный размер контента
summarize_threshold: 40000              # Порог для автоматической суммаризации
cache_ttl: 3600                         # Время жизни кеша в секундах
timeout: 30                             # Таймаут запросов
max_retries: 3                          # Максимум попыток retry
rate_limit: 5                           # Одновременные запросы к URL
system_prompt: "Ты — полезный ассистент."
mistral_system_prompt: "Ты — опытный ассистент, который улучшает ответы."
```

## Использование

```bash
# Базовый запрос
max-ai run "Привет, как дела?"

# Запрос с URL (автоматически прочитает страницу)
max-ai run "Что написано на этом сайте? https://example.com"

# Запрос с PDF
max-ai run "Прочитай этот PDF и сделай резюме https://example.com/document.pdf"

# Запрос с DOCX
max-ai run "Прочитай этот Word документ https://example.com/document.docx"

# Запрос с YouTube
max-ai run "О чем это видео? https://youtube.com/watch?v=example"

# Без кеширования
max-ai run "Ваш запрос" --no-cache

# Указать свой API ключ Cohere
max-ai run "Ваш запрос" --cohere-key YOUR_KEY

# Изменить TTL кеша (по умолчанию 3600 секунд)
max-ai run "Ваш запрос" --ttl 7200

# Посмотреть историю запросов
max-ai history

# Посмотреть историю (последние 5 записей)
max-ai history --limit 5

# Очистить кеш
max-ai cache-clear

# Проверить статус (с метриками)
max-ai status                 # Статистика за последнюю неделю
max-ai status --days 30       # Статистика за последний месяц
```

## FAQ

**Вопрос: Какой модель по умолчанию используется для Cohere?**
Ответ: По умолчанию используется модель `command-a-03-2025`.

**Вопрос: Можно ли использовать другие модели Cohere?**
Ответ: Да, можно указать модель через переменную окружения или параметр командной строки (планируется в будущих версиях).

**Вопрос: Как очистить кеш?**
Ответ: Используйте команду `max-ai cache-clear`.

**Вопрос: Как посмотреть историю запросов?**
Ответ: Используйте команду `max-ai history`.

**Вопрос: Почему не работает команда `max-ai` после установки?**
Ответ: Возможно, путь к скриптам не добавлен в PATH. См. раздел "Настройка PATH (Windows)".

**Вопрос: Можно ли использовать агент без API ключей?**
Ответ: Нет, для работы требуется хотя бы один API ключ (Cohere или Mistral).

**Вопрос: Какой формат ответов поддерживает агент?**
Ответ: Агент возвращает текстовые ответы. Он может извлекать текст из веб-страниц, PDF, DOCX, PPTX, XLSX, XLS, TXT, MD файлов и транскриптов YouTube.

**Вопрос: Как агент обрабатывает несколько URL в одном запросе?**
Ответ: Агент загружает все URL параллельно (используя асинхронные запросы) и объединяет их содержимое в контекст для модели.

**Вопрос: Есть ли ограничение на размер загружаемого контента?**
Ответ: Да, для каждого URL контент ограничивается 50 000 символов, чтобы избежать переполнения контекста модели.

**Вопрос: Как посмотреть статистику использования?**
Ответ: Используйте `max-ai status` или `max-ai status --days 30` для метрик за выбранный период.

**Вопрос: Как агент использует историю диалога?**
Ответ: Агент сохраняет историю запросов и ответов, и последние 10 сообщений добавляются в промпт для лучшего понимания контекста текущего запроса. Это позволяет модели учитывать предыдущие обмены при формировании ответа.

**Вопрос: Какие новые форматы файлов поддерживает агент?**
Ответ: Агент теперь поддерживает обработку PPTX (презентации), XLSX/XLS (электронные таблицы), TXT и MD (текстовые файлы) помимо существующих PDF, DOCX и YouTube транскриптов.

**Вопрос: Что происходит с большими документами?**
Ответ: Если контент превышает 40 000 символов, агент автоматически суммаризирует его через AI, чтобы уложиться в лимит контекста (50 000 символов). Это позволяет эффективно работать с большими PDF, DOCX и другими файлами.
